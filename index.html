Files & Directories 14
lsList directory contents. Use ls -la for hidden files + details
ls -lhSList files sorted by size (largest first), human-readable
cd /pathChange directory. cd ~ goes home, cd - previous dir
pwdPrint working directory (current path)
mkdir dirCreate directory. Use mkdir -p a/b/c to create nested dirs
rmdir dirRemove empty directory
rm fileDelete file. rm -rf dir removes directories recursively
cp src destCopy file. cp -r copies directories, cp -p preserves attrs
mv src destMove or rename a file/directory
touch fileCreate empty file or update timestamp of existing file
find . -name "*.txt"Find files by name in current directory and subdirectories
find . -mtime -7Find files modified in the last 7 days
treeDisplay directory structure as a tree
stat fileDisplay precise file details, size, and timestamps

Viewing & Editing 11
cat filePrint entire file to terminal. cat file1 file2 > file3 to merge
tac filePrint file in reverse (last line first)
less fileScroll through file. Press q to quit, /word to search
head -n 20 fileShow first 20 lines of a file
tail -n 20 fileShow last 20 lines. tail -f follows file in real-time

vim fileAdvanced text editor (Pro). Press i insert, :wq save/quit
wc -l fileCount lines. wc -w counts words, wc -c counts bytes
diff file1 file2Show differences between two files line by line
cmp file1 file2Compare two files byte by byte
file filenameDetect file type (text, binary, image, etc.)

whoamiDisplay the current logged-in username

⚡ Power User Terminal Mastery
Pipes (|)
Chain commands: ps aux | grep nginx | awk '{print $2}' passes output of the first as input to the next command.
Redirection (>, >>)
Use > to overwrite a file, >> to append. 2> redirects errors, &> redirects both stdout and stderr.
Ctrl Shortcuts
Ctrl+C kills, Ctrl+Z pauses, Ctrl+D logs out, Ctrl+L clears screen, Ctrl+R reverse searches your command history!
Wildcards (*, ?)
* matches any string of characters, ? matches a single char. Example: rm *.log deletes all log files instantly.
Background Jobs (&)
Append & to run in the background: ./script.sh &. Manage them smoothly with jobs, fg %1, or bg.
Tab & History Mastery
Press Tab to auto-complete commands. Type !540 to run command #540 from history, or !! to run the last command with sudo !!
Variables & Aliases
alias ll='ls -alF' creates a shortcut. Set variables: export VAR="hello". Save these in ~/.bashrc or ~/.zshrc to make them permanent.
Brace Expansion
Create files quickly: touch file_{1..5}.txt makes file_1 to 5. mkdir -p src/{components,utils} scaffolds a project structure instantly.
