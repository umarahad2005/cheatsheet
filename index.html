Linux Commands

$ your complete reference cheatsheet

📁
Files & Directories 12
lsList directory contents. Use ls -la for hidden files + details
ls -laLong listing with hidden files, permissions, size, owner
cd /pathChange directory. cd ~ goes home, cd .. goes up one level
pwdPrint working directory — shows your current path
mkdir dirCreate directory. Use mkdir -p a/b/c to create nested dirs
rmdir dirRemove empty directory
rm fileDelete file. rm -rf dir removes directories recursively (careful!)
cp src destCopy file. cp -r to copy directories recursively
mv src destMove or rename a file/directory
touch fileCreate empty file or update timestamp of existing file
find / -name xFind files by name. find . -type f -name "*.txt" finds all txt files
treeDisplay directory structure as a tree (install if missing)
👁️
Viewing Files 9
cat filePrint entire file to terminal. Good for small files
less fileScroll through file. Press q to quit, /word to search
more filePage through file (older, less features than less)
head -n 20 fileShow first 20 lines of a file
tail -n 20 fileShow last 20 lines. tail -f follows file in real-time
wc -l fileCount lines. wc -w counts words, wc -c counts bytes
grep "text" fileSearch for text. grep -r "text" . searches recursively
diff file1 file2Show differences between two files line by line
file filenameDetect file type (text, binary, image, etc.)
🔒
Permissions 8
chmod 755 fileSet permissions: 7=rwx, 5=r-x, 4=r--. Owner/group/others
chmod +x fileAdd execute permission to file (make it runnable)
chown user fileChange file owner. chown user:group file changes both
chgrp group fileChange group ownership of a file
umask 022Set default permission mask for new files
sudo commandRun command as superuser (root)
su usernameSwitch to another user account
whoamiDisplay the current logged-in username
⚙️
Process Management 10
ps auxList all running processes with PID, CPU, memory usage
topReal-time process viewer. Press q to quit, k to kill
htopInteractive process viewer (nicer than top, may need install)
kill PIDKill process by PID. kill -9 PID force kills immediately
killall nameKill all processes with a given name
bgSend paused process to background
fgBring background process to foreground
jobsList background and paused jobs in current session
nohup cmd &Run command in background, immune to hangups
nice -n 10 cmdRun command with lower priority (10 = lower, -10 = higher)
🌐
Networking 11
ping hostTest connectivity to a host. Ctrl+C to stop
ifconfigShow network interface info (use ip addr on modern systems)
ip addrShow all IP addresses and network interfaces
netstat -tulpnList open ports and listening services
ss -tulpnModern replacement for netstat — shows socket stats
curl URLMake HTTP requests. curl -O URL downloads a file
wget URLDownload files from web. Supports recursive downloads
ssh user@hostConnect to remote server via SSH
scp src user@h:destSecurely copy files to/from remote host
traceroute hostShow network hops to reach a destination
nslookup domainQuery DNS records for a domain name

Disk & System Info 10
df -hShow disk space usage in human-readable format
du -sh dirShow total size of a directory. du -sh * sizes of all items
free -hShow RAM and swap usage in human-readable format
uname -aShow system info: kernel, hostname, architecture
uptimeShow how long system has been running and load avg
lscpuDisplay CPU architecture and core information
lsblkList block devices (disks and partitions)
mount /dev/x /mntMount a device to a directory
umount /mntUnmount a mounted device or filesystem
dmesg | tailView kernel log messages (hardware events, errors)
✂️
Text Processing 10
grep -i "pat" fCase-insensitive search. -v inverts (shows non-matching)
sed 's/a/b/g' fReplace 'a' with 'b' globally in file. -i edits in place
awk '{print $1}' fPrint first column. awk -F: '{print $1}' uses : as delimiter
sort fileSort lines alphabetically. sort -n sorts numerically
uniq fileRemove duplicate adjacent lines. Use after sort
cut -d: -f1 fCut field 1 from each line using : as delimiter
tr 'a-z' 'A-Z'Translate characters — converts lowercase to uppercase
echo "text"Print text to terminal. Use with > to write to file
tee fileRead stdin and write to both file and stdout simultaneously
xargsBuild commands from stdin. find . -name x | xargs rm

Package & Shell 11
apt install pkgInstall package (Debian/Ubuntu). Use sudo apt install
apt updateRefresh package lists from repositories
apt upgradeUpgrade all installed packages to latest versions
apt remove pkgUninstall a package. apt purge also removes config files
yum install pkgInstall package on RHEL/CentOS/Fedora systems
historyShow command history. !n runs command #n again
alias ll='ls -la'Create a shortcut for a command
export VAR=valSet environment variable for current session
source ~/.bashrcReload shell config without restarting terminal
which cmdShow full path of a command executable
man commandOpen manual page for any command. Press q to quit
⚡ Power User Tips
Pipes
Chain commands with |: ps aux | grep nginx | awk '{print $2}' — output of one becomes input of next.
Redirection
Use > to write to file, >> to append, 2> for errors, &> for both stdout+stderr.
Ctrl Shortcuts
Ctrl+C kills, Ctrl+Z pauses, Ctrl+D logs out, Ctrl+L clears screen, Ctrl+R searches history.
Wildcards
* matches anything, ? matches one char. Example: rm *.log deletes all .log files.
Background Jobs
Append & to run in background: ./script.sh &. Use jobs and fg %1 to manage them.
Tab Completion
Press Tab to auto-complete commands and paths. Double-tap Tab to see all options when ambiguous.
